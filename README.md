# cfop-notebookLM
Desafio para a DIO usando o notebookLM para criar uma IA especialista em resolver o cubo magico pelo método cfop


Este projeto foi feito com intuito de resolver um desafio para a plataforma DIO, nele eu devo criar uma IA com uma especialidade usando fontes diversas. O tema que eu escolhi para fazer a minha IA é CFOP. O CFOP é um método avançado para resolver o cubo mágico, usado por alguns dos melhores no mundo de speedcubing. O método possui niveis de aprendizado, desde mais avançados ao mais iniciante. Os mais iniciantes exigem mais etapas, porém menos algoritmos a ser aprendidos, que os mais avançados e a IA cobrirá como aprender o CFOP desde iniciante a mais avançado. O método mais avançado consiste em 4 etapas: cruz, F2L, OLL e PLL.  O motivo pelo qual eu escolhi esse assunto é que eu estou tentando melhorar no cubo mágico e eu apenas sei CFOP iniciante e essa ia pode me ajudar a aprender o método CFOP mais avançado.
